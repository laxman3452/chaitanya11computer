The Standard Template

// 2. Main Function - execution starts here

    // 3. Statements / Executable Code

    // 4. Return Statement

Input and Output (I/O) & Format Specifiers

To communicate with the outside world, C relies on functions packaged inside <stdio.h>. Because C is statically typed, you must use Format Specifiers as placeholders to tell printf() and scanf() exactly how to interpret data types traveling through streams.

Essential Format Specifiers Reference

Specifier Data Type Usage Example Output Representation
%d or %i Integer int age = 25; Base-10 signed whole numbers (e.g., 25)
%f Float (Decimal numbers) float price = 12.99; Fractional values (defaults to 6 decimal places: 12.990000)
%.2f Float (Formatted) float price = 12.99; Limits layout representation to 2 decimal places (e.g., 12.99)
%c Character char grade = 'A'; A single letter, digit, or symbol (e.g., A)
%s String (Char Array) char name[] = "John"; Sequences of characters until a null-terminator is hit

Detailed Real-World Use Case: Automated Billing Engine

Imagine writing software for a physical store kiosk. The app must prompt the employee for an item name (string), quantity purchased (integer), and unit cost (floating point decimal), process calculations, and present a formatted checkout ticket.

#include <stdio.h>

int main() {
    char item[30];
    int qty;
    float unitPrice, total;

    // 1. Reading User Input via Scanf passing addresses
    printf("Enter Item Name: ");
    scanf("%s", item); // No & needed for arrays/strings!

    printf("Enter Quantity & Unit Price: ");
    scanf("%d %f", &qty, &unitPrice); // & essential for primitive values

    // Processing Logic
    total = qty * unitPrice;

    // 2. Displaying clean structured invoice receipts
    printf("\n--- INVOICE RECEIPT ---\n");
    printf("Item: %s\n", item);
    printf("Qty:  %d units @ $%.2f each\n", qty, unitPrice);
    printf("Total Due: $%.2f\n", total);

    return 0;
}

Data Types in C

C is a statically typed language. You must explicitly declare what box size and formatting type a variable belongs to before storing data in it.

Syntax Guides

// 1. Integer (Whole numbers)
int speed = 85;

// 2. Array (Homogeneous continuous sequence)
int scores[3] = {90, 85, 95};

// 3. String (Array of characters ending with a null terminator '\0')
char greeting[] = "Hello";

Conditional Statements

Conditionals provide logic branching, allowing programs to make decisions based on runtime values using boolean logic patterns.

Syntax

if (condition) {
    // Runs if true
} else if (another_condition) {
    // Runs if secondary condition is true
} else {
    // Runs if all conditions fail
}

Loops in C

Loops allow code blocks to be repeated until a specific loop termination condition is met.

Syntax

// For Loop: Ideal when iterations count is predetermined
for (int i = 0; i < count; i++) { ... }

// While Loop: Dynamic evaluation checking condition at each step start
while (condition) { ... }
